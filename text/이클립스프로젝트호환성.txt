이클립스 호환성 체크

프로젝트 import 후

첫번쨰방법
1. 프로젝트 우클릭 Build Path(Configure Build Path) - Java Build Path(Libraries 탭) - Tomcat 버전 확인
JRE System library 버전 확인 (추가시 installed JREs)

2. 이클립스 Preferences - Server - runtime Environment - 톰캣 Edit - JRE 버전 확인

3. 다시 Build Path(Configure Build Path) - Project Facets - Java 버전 및 Dynamic Web Module 확인

셋다 동일한 Java 버전으로 맞춰주기 



2번째 방법
이클립스 - Window - Show View - Other - Markers 후 콘솔창 옆 Markers에서 에러나는 부분 Quick 솔루션 클릭

*이클립스 최신버전은 자바 1.8 이상 버전이 필요
*자바를 여러개 보유 가능
