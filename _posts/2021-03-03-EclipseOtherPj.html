Eclipse에서 다른 프로젝트 클래스를 참조하는 방법

대규모 프로젝트의 경우 여러 프로젝트로 나누어 관리하는 경우 다른 프로젝트 클래스들을 참조하기 위한 방법을 알아보자.

  • 프로젝트의 properties 클릭 or 단축키 Alt+Enter
  • 왼쪽 tab의 Java Build Path
  • 상단 tab의 Projects 클릭
  • Add 클릭 후 원하는 프로젝트 선택 후 Apply
